# Restaurant Priority App

Live on Vercel! 🎉

## URLs

- **Main page:** https://restaurant-deploy-nine.vercel.app/
- **Priority Editor:** https://restaurant-deploy-nine.vercel.app/priority-editor
- **GitHub:** https://github.com/AtomicRasta/restaurant-priority

## Updating

1. Edit files locally
2. Commit and push to GitHub:
   ```bash
   git add .
   git commit -m "Update"
   git push
   ```
3. Vercel auto-deploys on push

## Files

- `index.html` - Main weekly availability view
- `priority-editor.html` - Drag-drop prioritization interface
- `restaurant-priority.json` - Restaurant data (editable via priority editor)
- `vercel.json` - Deployment config
